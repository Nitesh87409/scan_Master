# Document Scanner + PDF Toolkit Roadmap

## Vision

Build a fast, modern, offline-first Flutter app.

### Goals

-   Flutter only
-   Android first
-   No AI APIs
-   No paid SDKs/libraries
-   No backend/server
-   No subscriptions
-   Offline-first
-   Material 3
-   Light & Dark theme
-   Minimal ads

## Phases

### Phase 1

-   Camera scan
-   Gallery import
-   Crop
-   Rotate
-   Brightness
-   Contrast
-   Save PDF
-   Save JPG
-   Merge PDF
-   Split PDF
-   Share

### Phase 2

-   Compress PDF
-   Password protect
-   Unlock
-   Watermark
-   Image to PDF
-   Print

### Phase 3

-   Signature
-   QR Scanner
-   QR Generator
-   Batch Scan
-   Favorites
